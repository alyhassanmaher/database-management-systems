# Database Management Systems — Final Project

A college database project covering database design, SQL querying, and data management.

## Contents

| File | Description |
|------|-------------|
| `finaldatabase.zip` | Full project — schema, SQL scripts, and supporting files |

## Topics Likely Covered

- Entity-Relationship (ER) Diagram design
- Database normalization (1NF, 2NF, 3NF)
- SQL DDL — `CREATE`, `ALTER`, `DROP`
- SQL DML — `INSERT`, `UPDATE`, `DELETE`, `SELECT`
- Joins, subqueries, stored procedures
- Indexing and query optimization

## How to Use

Extract the zip file to access all database files:

```bash
unzip finaldatabase.zip
```
